HDR check

NPM package to check hdr support in css (browser and hardware)

  • Check if browser supports HDR
  • Check if hardware supports HDR
npm i isit-hdr-ready
  • GitHub
  • NPM

Usage

import isitHDRReady from 'isit-hdr-ready';

Example

code

import isitHDRReady from 'isit-hdr-ready';
       
        const hdrReadyMessage = document.querySelector(".hdr-ready-message");
        if (isitHDRReady()) {
          hdrReadyMessage.classList.remove("hidden");
        }
